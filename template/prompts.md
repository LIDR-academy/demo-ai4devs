# Prompt utilizando ChatGPT

Como programador, utilizando las mejores prácticas, necesito el código de una página web en html que apoyada de un script (script.js) haga lo siguiente:
- La página web debe tener un input para poder introducir texto
- Debe existir un botón, el cual se debe activar cuando el input tenga más de 3 caracteres
- el botón debe estar a la derecha del input
- Cuando el botón sea pulsado (si está activo) debe ejecutarse la función reverseString que debes definir en el archivo script.js. Esta función es la encargada de dar vuelta el texto entregado en el input, por ejemplo si se entrega el texto "hola mundo", debe retornar "odnum aloh"
- si el botón está deshabilitado haz que salga un tooltip o un title que diga la razón
- muestra el resultado abaja de del input y del botón con el texto
"Texto invertido al presionar botón: <resultado>"
- finalmente agrega abajo el texto producto de la función reverseString cada vez que cambia el valor del input de la siguiente forma:
"Texto invertido al cambiar el input: <resultado>"
